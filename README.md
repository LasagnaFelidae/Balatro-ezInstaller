# Balatro-ezInstaller
Easy user-friendly installer option to setup Balatro Modding (Lovely+SMODS) for Windows

Selects latest by default, allows the option to have a different location for the Mods folder, incase you dont have much space in your C: drive.

## How to install

1. go to releases or click [this link](https://github.com/LasagnaFelidae/Balatro-ezInstaller/releases/latest)
2. download the exe
3. run the installer
4. done 👍

## Notes

* windows only
* you need internet
* make sure balatro is installed and not running
